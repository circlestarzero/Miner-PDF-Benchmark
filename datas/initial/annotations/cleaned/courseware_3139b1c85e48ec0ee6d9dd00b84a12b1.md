1.结构章法

$$
\left\{\begin{array}{l}
\text { 古体诗: 以自然段为层 } \\
\text { 近体诗 }\left\{\begin{array}{l}
\text { 绝句:前两句写景(次),后两 } \\
\text { 句抒情(第四句最重要) } \\
\text { 律诗:前四句写景(次),后四句 } \\
\text { 抒情(主) }
\end{array}\right\} \text { 有的诗讲究 承转合 }
\end{array}\right.
$$

$$
\begin{aligned}
& \text { 词 }\left\{\text { 上片(闺) }\left\{\begin{array}{l}

\text { 尾句过渡 }
\end{array}\right\}\right. \text { 写景(次) } \\
& \text { (下片(闺):抒情(主) }
